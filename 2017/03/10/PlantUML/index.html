PlantUML简单介绍
发布于：2017-03-10 17:49:14 标签：/ uml / PlantUML / 访问：

PlantUML

PlantUML 是快速创建UML图形的组件, 一般在其他软件里使用,如: Ecplise,NetBenas, Emacs, Word和Intellijidea等.

PlantUML支持的图形有:

  • sequence diagram
  • use case diagram
  • class diagram
  • activity diagram
  • component diagram
  • state diagram
  • object diagram
  • wireframe graphical interface

注释:

  • 单行注释: 以单引号开头,即注释一行
  • 多行注释: 以/‘开头,以’/结尾

参与者申明:

可使用participant关键词, 亦可使用以下参与者分类关键词进行申明

  • actor
  • boundary
  • control
  • entity
  • database

Sequence Diagram

用: ->, –>, <-, <–来绘制参与者之间的消息.
